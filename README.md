# Project-potato
### Simple android game made in Unity3D

If you want to govern the Big Potato Empire or just want to be best in potatoes like Technoblade in Hypixel Skyblock, this game is completely for you. You can plant potatoes and sell them or keep them for yourself. You are able to born children, hire workers and make them work for you. They will plant potatoes exclusively for your kingdom and improve it.

Features:
- Gathering potatoes
- Buying upgrades for yourself
- Buying lands ( like different planets)
- Buying upgrades for lands
- Birthing children
- Growing children
- Teaching children how to be a worker
- Hiring Workers
- Getting potatoes from Workers

If you want to play, you can visit google play site of it: [Google Play](https://play.google.com/store/apps/details?id=com.PotatoCompany.ProjectPotato)
